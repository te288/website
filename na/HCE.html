熱伝導方程式のシミュレーション


1.支配方程式

一様な物質でできている物体内部の温度 $T(x,y,z,t)$ に関する熱拡散方程式は，\(\kappa\)を熱拡散係数として次のように表される(参考)。
$$ \frac{\partial T}{\partial t} = \kappa\nabla^2 T\tag{1} $$
式$\left(1\right)$を有限体積法で離散化し、陽解法で解いてみる。

3.プログラム

離散化した式を計算するMATLABプログラム。収束判定は行わず，規定の回数計算した時，whileループを抜ける。MATLABはループ計算が遅いので，高速化したプログラムを公開する予定。
clear; close all;
%% difinition of variables arrays
% definition -------------------------------------------------------
Lx = 1.0; % length of x-direction [m]
Ly = 1.0; % length of y-direction [m]
nx = 50; % number of grid in x-direction [m]
ny = 50; % number of grid in y-direction [m]
dx = zeros(nx+2,1); dx(2:nx+1) = Lx / nx; % size of dx[m]
dy = zeros(ny+2,1); dy(2:nx+1) = Ly / ny; % size of dy[m]
TL = 300; % low temperature [K]
TH = 500; % high temperature [K]
crit = 0.01; % criteria for convergence [K]
a = 1.0; % heat diffusivity [m2/s]
dt = 0.5*(1/dx(2)^2 + 1/dy(2)^2)^-1/a; %dt
NO = 10000; % time step
ele = (nx+2)*(ny+2); % number of elements including boudary
n = 0; % Loop counter [-]
x = zeros(nx+2,1); % Coordinate point - X
y = zeros(ny+2,1); % Coordinate point - Y
for s = 2:nx+2
x(s) = x(s-1) + 0.5*(dx(s)+dx(s-1));
end
for s = 2:nx+2
y(s) = y(s-1) + 0.5*(dy(s)+dy(s-1));
end
% temperture
t_old = zeros(ny+2, nx+2); % temperture matrix
t_new = zeros(ny+2, nx+2); % temperture matrix
t_old(:,1) = TL;
t_old(:,nx+2) = TL;
t_old(ny+2,:) = TH;
t_old(1,:) =TL;
t_new(:,:) = t_old(:,:);
%t(:,:) = t_old(:,:); t_new(:,:) = t_old(:,:);
% coefficients
K = zeros(ny+2, nx+2); %coefficients matrix(ny+2, nx+2)
co = zeros(ny+2, nx+2);
ct = zeros(ny+2, nx+2);
cn = zeros(ny+2,1);
cs = zeros(ny+2,1);
cw = zeros(1,nx+2);
ce = zeros(1,nx+2);
% Set value of valiables
% ct
ct(:,:) = 1.0;
ct(:,:) = dx(2)*dy(2)/dt.*ct(:,:);
% cn, cs
for j = 2:ny+1
for i = 2:nx+1
cn(j) = a*dx(i)/(0.5*(dy(j)+dy(j+1)));
cs(j) = a*dx(i)/(0.5*(dy(j)+dy(j-1)));
end
end
% cw, ce
for j = 2:ny+1
for i =2:nx+1
cw(i) = a*dy(j)/(0.5*(dx(i-1)+dx(i)));
ce(i) = a*dy(j)/(0.5*(dx(i+1)+dx(i)));
end
end
% co
for j = 2:ny+1
for i = 2:nx+1
co(j,i) = cw(i)+ce(i)+cn(j)+cs(j);
end
end
%% Simulation ----------------------------------------------------
while true
for j = 2:ny+1
for i = 2:nx+1
t_new(j,i) = (cs(j) *t_old(j-1,i)...
+cn(j) *t_old(j+1,i)...
+ce(i) *t_old(j,i+1)...
+cw(i) *t_old(j,i-1)...
+(ct(j,i)-co(j,i))*t_old(j,i)...
)/ct(j,i);
end
end
t_old(:,:) = t_new(:,:);
n = n+1;
if n == NO
break
end
end
%% Show result -----------------------------------------------------
[X,Y] = meshgrid(x,y);
[M, c] = contourf(X,Y,t_new, 100);
set(c,'Linestyle','none');
colorbar
title('熱伝導方程式の数値解（陽解法）');
xlabel('x[m]');
ylabel('y[m]');
colormap('turbo')
